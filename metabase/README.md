# Metabase

## Описание
Анализ банковских транзакций за год с использованием Metabase.

## Предобработка

Перед загрузкой в Metabase данные были обработаны на Python (pdfplumber, pandas)

- данные  извлечены из банковской PDF-выписки и преобразованы в формат (CSV)
- суммы операций приведены к числовому виду
- удалены символы и знаки валют

### Категории

Была выполнена базовая группировка операций:
- расходы распределены по исходным категориям (некоторые категории распределяла вручную, потому что большое кол-во переводов это реальная оплата ( например переводы по СПБ сложно отнести к категории не вручную))
- оставшие переводы(которые не смогла структурировать) были объединены в категорию «Прочее»


## Загрузка и анализ

После предобработки данные были загружены в Metabase и использовались без дополнительных преобразований

## Содержимое
- SQL-запросы для анализа доходов и расходов
- визуализации (кэшбек, категории, Sankey и тд)
- фильтры по месяцу и категории
- кросс-фильтры

## Dashboard

Dashboard включает анализ:
- доходов и расходов по месяцам
- структуры расходов по категориям
- кэшбека
- кумулятивную сумму (месяцы, кварталы)
- распределения расходов (Sankey, Pie)

Добавлены фильтры по месяцу и категории, а также кросс-фильтрация графиков.
